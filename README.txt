This app is supposed to be a free alternative and bring together Candle and Setlog, two long-distance relationship apps.

Current state of the project:
- Code is connected with Supabase to handle the database
- Sign in is completed and we can create shared spaces with other users
- There are a few categories of questions from which to choose from and you can create chats around these questions

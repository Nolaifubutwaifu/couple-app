Design Guide

Mood:
Soft, romantic, polished, warm, slightly playful.

Colors:
- Background: warm cream
- Card: soft white
- Primary text: deep navy
- Secondary text: muted gray-blue
- Primary accent: blush pink
- Secondary accent: lavender
- Optional accent: soft mint green
- Success: muted green

Shape:
- Cards: very rounded
- Buttons: pill-shaped or rounded
- Images: rounded corners
- Tabs: pill segmented controls

Shadows:
- Soft and subtle
- No harsh black shadows

Typography:
- Big bold page titles
- Small uppercase section labels
- Calm readable body text
- Avoid too many font sizes

Interaction hierarchy:
- One clear main action per screen
- Secondary actions should be quieter
- Avoid too many equally loud cards

Do:
- Use reusable CSS variables
- Use reusable card/button classes
- Keep layout mobile-first
- Keep spacing consistent

Do not:
- Use harsh colors
- Use default browser buttons
- Make everything pink
- Add too many borders
- Break existing logic
